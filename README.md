# Video Scanner (Client-Only)

A lightweight, secure, and fast video content safety scanner using Google Gemini 2.5 Flash.

## Features
- **Client-Side Only**: No backend server required.
- **Secure**: Your API key is stored locally in your browser (LocalStorage) and never sent to our servers.
- **Direct Analysis**: Sends YouTube URLs directly to Google's Gemini API.
- **Fast**: Built with Vue.js and Tailwind CSS for instant feedback.

## How to Use
1. Get a [Google Gemini API Key](https://aistudio.google.com/app/apikey).
2. Open the app.
3. Paste your API Key (it will be saved for next time).
4. Paste a YouTube URL.
5. Click "Scan".

## Deployment
This project is ready for static hosting.
- **Vercel**: Drag and drop this folder or connect your GitHub repo.
- **GitHub Pages**: Enable pages in settings.
